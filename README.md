# Flask API

A simple Flask API project built using Python and Flask.

## Project Structure

* `main.py` - Main application file
* `flask.py` - Flask-related code
* `plog.py` - Logging functionality

## Requirements

* Python 3.x
* Flask

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Diyaranidhauskar8/api.git
   ```

2. Navigate to the project folder:

   ```bash
   cd api
   ```

3. Install dependencies:

   ```bash
   pip install flask
   ```

## Run the Project

```bash
python main.py
```
